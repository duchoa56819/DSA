## 📋 Yêu Cầu
- Python 3.8+
- Trình duyệt web (cho visualizer)

## 📖 Cách Học Hiệu Quả
1. **Đọc lý thuyết** (`theory.md`) trước
2. **Đọc và chạy code** minh họa
3. **Tự viết lại** code mà không nhìn
4. **Làm bài tập** ở cuối mỗi chương
5. **Dùng visualizer** để hiểu trực quan
